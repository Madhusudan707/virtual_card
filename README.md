#### Get started

1. **Clone the repo:**
```bash
  $ git clone https://github.com/Madhusudan707/virtual_card.git
```
1. **Change the Directory:**
```bash
  $ cd virtual_card
```
3. **Install required node modules:**
```bash
  $ npm install
```
4. **Start the dev server:**
```bash
  $ npm start
```
Open the `localhost` link in the browser.
Happy hacking!
